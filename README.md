## Current project: StatementLens

[StatementLens](https://github.com/matthewhurt12/statementlens) inspects a bank CSV in one
offline-capable HTML file. It has no account, backend, analytics, saved history, or transaction
upload.

- [Try the public app](https://matthewhurt12.github.io/statementlens/)
- [Download v0.1.0](https://github.com/matthewhurt12/statementlens/releases/tag/v0.1.0)
- [Join the public-beta discussion](https://github.com/matthewhurt12/statementlens/discussions/4)

The goal is straightforward: keep building, publish the work, listen to real users, record
what succeeds and fails, and improve the next project.
